1) cd
2) cd~ - возвращает в домашний каталог
   cd .. - возвращает назад в каталоге
3) rm - это удаление файлов и директорий 
4) cd - это втроенная оболочка 
5) cp - это копрование файлов 
6) mv - это перемещение файлов и для переименования файлов
7) cp старое_название новое_название
8) mv старое_название новое_название
9) -R обозначает копирование и перемещение
10) -r удаление файлов
11) git - это система контроля версий, поворляет отсдеживать именения в исходном коде
12) got init - это команда, который инициализирует новый репозиторий
13) Чтобы создать новую ветку в Git, используйте команду "git branch имя_ветки"                                      
14) git checkout имя_ветки
15) Слияние (merge)- это процесс объединения содержимого одной ветки с другой.
16) "git rm" - это команда Git для удаления файла из репозитория.
17) "git revert" -  отменяет определенный коммит, создавая новый коммит, который отменяет изменения
